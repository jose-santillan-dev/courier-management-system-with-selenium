# Courier Management System with Selenium

This project is a web-based Courier Management System developed using ASP.NET.  
It includes a functional web application and automated end-to-end UI tests implemented with Selenium WebDriver.

## Features
- Courier registration and management
- Basic shipment tracking functionality
- Web-based user interface built with ASP.NET
- Automated UI testing using Selenium WebDriver

## Technologies Used
- ASP.NET (.NET)
- Selenium WebDriver
- C#
- Visual Studio
- MySql


## Project Structure
Courier.Web/ -> ASP.NET web application
Courier.Tests/ -> Selenium automated tests

## Automated Testing
The Selenium test project validates key user flows of the web application, ensuring that critical functionalities behave as expected.

## Purpose
This project was developed as part of my academic training and later refined for professional portfolio use, focusing on clean structure, automation, and real-world practices.

## Database ⚠️
This project uses MySQL as the database engine.

To run the project locally:
- A MySQL server is required
- The connection string must be configured in the application settings

The database is not included in this repository for security and portability reasons.
